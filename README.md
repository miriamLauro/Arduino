# 🚦 Arduino Projects Collection  

Una raccolta di progetti base realizzati con **Arduino IDE** e simulati su **Tinkercad**.  

---

## 📦 Progetti inclusi  

### 💡 LED Blink  
Un LED che lampeggia a intervalli regolari.  

**✔️ Concetti base:**
- Output digitale  
- Uso della funzione `delay()`  


---

### 🚦 Semaforo  
Simulazione di un semaforo con LED rosso, giallo e verde.  

**✔️ Concetti base:**
- Sequenze temporizzate  
- Gestione di più pin  
 

---

### 🚥 4 Semafori (Incrocio)  
Simulazione di un incrocio con quattro semafori coordinati.  

**✔️ Concetti base:**
- Coordinazione tra più semafori  
- Gestione del traffico simulato  

---

## 🚶 Semaforo con Pulsante  
Semaforo pedonale attivato tramite pulsante.  

**✔️ Concetti base:**
- Input digitale (pulsante)  
- Strutture condizionali (`if`)  
  

---

## 🛠️ Tecnologie utilizzate  
- 🧠 Arduino IDE  
- 🔧 Tinkercad Circuits
